# 🧠 Intelligent Code Analysis and Optimization System (Gemini AI)

## 📌 Overview

The AI Code Debugger is an intelligent developer tool built using Streamlit and Google Gemini AI.
It helps developers analyze, debug, optimize, and understand code efficiently using Generative AI.

This project acts as a virtual coding assistant capable of handling multiple developer tasks in one place.

---

## ❗ Problem Statement

Software development often involves repetitive and time-consuming tasks such as debugging errors, understanding unfamiliar code, optimizing performance, and identifying security vulnerabilities.

Many developers, especially beginners, struggle with:

* Identifying the root cause of errors
* Understanding complex codebases
* Writing optimized and efficient code
* Ensuring code security and reliability
* Generating proper test cases

Existing tools either focus on a single functionality (like debugging or linting) or require manual effort and expertise, making the development process slower and less efficient.

---

## 💡 Proposed Solution

To address these challenges, we developed an **AI-Powered Code Debugger & Multi-Functional Code Assistant** using Generative AI.

The system leverages **Google Gemini AI** to provide an integrated solution that:

* 🔍 Automatically detects errors and suggests fixes
* 💡 Explains code in simple, human-readable language
* ⚡ Optimizes code for better performance
* 🔐 Identifies potential security vulnerabilities
* 🧪 Generates test cases for improved reliability
* 🔄 Converts code between multiple programming languages

The application is built with **Streamlit**, providing an intuitive and interactive interface where users can:

* Upload files or paste code
* Perform multiple analyses with a single click
* Get instant AI-generated insights

---

## 🎯 Key Benefits

* Reduces manual debugging effort
* Improves code quality and efficiency
* Enhances learning for beginners
* Acts as an all-in-one AI coding assistant
* Saves developer time and increases productivity

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/YOUR_USERNAME/ai-code-debugger.git
cd ai-code-debugger
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Add API Key

Create `.env` file:

```
GOOGLE_API_KEY=your_api_key_here
```

### 4️⃣ Run Application

```
streamlit run app.py
```
